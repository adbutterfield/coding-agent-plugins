# Debating Code Reviews - Reference Index

This directory contains detailed reference materials for the adversarial debate code review skill.

## Reference Files

| File | Purpose |
|------|---------|
| [reviewer-personas.md](reviewer-personas.md) | Complete persona definitions and spawn prompts for all 4 reviewers (Tracer, Architect, Breaker, Prosecutor) |
| [debate-protocol.md](debate-protocol.md) | Round structure, messaging patterns, severity definitions, timing guidance, and anti-patterns |
| [synthesis-template.md](synthesis-template.md) | Final report template, triage criteria, and verdict guidelines |

## How to Use

1. **Setting up a review team** - See `reviewer-personas.md` for the spawn prompts to include when creating each agent
2. **Running the debate** - Follow the round structure in `debate-protocol.md`
3. **Producing the report** - Use the template in `synthesis-template.md` for the final consolidated output
