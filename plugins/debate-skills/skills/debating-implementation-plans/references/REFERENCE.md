# Debating Implementation Plans - Reference Index

This directory contains detailed reference materials for the adversarial debate implementation planning skill.

## Reference Files

| File | Purpose |
|------|---------|
| [planner-personas.md](planner-personas.md) | Complete persona definitions and thinking styles for all 4 planning agents (Scout, Planner, Critic, Prosecutor) |
| [planning-debate-protocol.md](planning-debate-protocol.md) | Round structure, messaging patterns, discovery coordination, and anti-patterns |
| [plan-template.md](plan-template.md) | Final plan document format designed for direct consumption by coding agents |

## How to Use

1. **Setting up a planning team** - See `planner-personas.md` for the spawn prompts to include when creating each agent
2. **Running the debate** - Follow the round structure in `planning-debate-protocol.md`
3. **Producing the plan** - Use the template in `plan-template.md` for the final consolidated output
